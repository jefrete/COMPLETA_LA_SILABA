# https://jefrete.github.io/juego_hm/
Juego de Sílabas

Aplicación web sencilla para que los niños de primaria practiquen las sílabas.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Lee la palabra incompleta y elige la sílaba que falta.
3. Completa todos los niveles y mejora tu puntuación.
